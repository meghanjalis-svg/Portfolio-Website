# Cinematic Developer Portfolio Website

A modern cinematic portfolio website built to showcase my projects, skills, certifications, and development journey through immersive interactions and scroll-based animations.

This portfolio combines smooth motion design, futuristic UI aesthetics, and responsive web technologies to create an engaging digital experience rather than a traditional static portfolio.

## ✨ Features

* Fullscreen cinematic hero section with background video
* Smooth scroll-triggered animations using Framer Motion & GSAP
* Interactive project showcase with animated transitions
* Modern glassmorphism UI with dynamic lighting effects
* Responsive design optimized for all devices
* Animated certifications & skills section
* Splash screen / preloader experience
* Premium hover interactions and micro-animations
* Fully customizable and scalable architecture

## 🛠️ Tech Stack

* React.js
* Next.js
* Tailwind CSS
* Framer Motion
* GSAP ScrollTrigger
* Lenis Smooth Scrolling
* Lucide React

## 🚀 Purpose

This portfolio was designed to reflect both my technical skills and creative approach to frontend development by blending modern web technologies with cinematic user experiences.

## 📌 Note

The content, projects, certifications, and experiences displayed in this portfolio are based on my personal academic and professional journey.
